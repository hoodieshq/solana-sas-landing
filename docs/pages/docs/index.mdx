## Installation

Install the SAS library within your project:

### Typescript / Javascript

```bash [npm]
npm i sas-lib
```

```bash [yarn]
yarn add sas-lib
```

```bash [pnpm]
pnpm i sas-lib
```

### Rust

```bash
cargo add solana-attestation-service-client
```
