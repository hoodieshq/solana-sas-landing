## Manual Installation

To manually add the SAS library to your project, you can use the following commands:

### Typescript / Javascript

```bash [npm]
npm i sas-lib 
```

```bash [yarn]
yarn add sas-lib 
```

```bash [pnpm]
pnpm i sas-lib 
```

### Rust

```bash
cargo add solana-attestation-service-client
```
